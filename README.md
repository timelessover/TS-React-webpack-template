# 从零搭建一个TS+React脚手架模板


## 体验

> 下载项目：
>
> git clone git@github.com:timelessover/TS-React-webpack-template.git
>
> 安装依赖：
>
> yarn add
>
> 启动项目：
>
> yarn start
